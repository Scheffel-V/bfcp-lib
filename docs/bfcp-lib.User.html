Class: User

User(userId, conferenceId)

This class is a abstract representation of a User in the BFCP environment, and is used as the primary interface to this library. A User receive and returns BFCP messages totally in the binary form, as like when receiving from TCP/UDP sockets, so the application who utilize it doesn't need to know anything about the codification and decodification of BFCP messages.
